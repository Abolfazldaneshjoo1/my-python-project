پروژه پیش‌بینی قیمت خانه - House Price Prediction Extra
-----------------------------------------------------

📌 توضیحات پروژه:
این پروژه با استفاده از الگوریتم رگرسیون خطی در پایتون، قیمت خانه‌ها را پیش‌بینی می‌کند. مدل بر اساس ویژگی‌های مختلف خانه‌ها آموزش دیده است.

📁 فایل‌های پروژه:
- house_price_prediction_extra.py (کد اصلی)
- housing.csv (دیتاست اصلی)
- requirements.txt (نیازمندی‌ها)

⚙️ نیازمندی‌های سیستم:
- پایتون 3.8 یا بالاتر
- کتابخانه‌های مورد نیاز:
  pandas==1.3.4
  numpy==1.21.2
  matplotlib==3.4.3
  seaborn==0.11.2
  scikit-learn==0.24.2

🚀 نحوه اجرا:
1. تمام فایل‌های پروژه را در یک پوشه دانلود کنید
2. ترمینال را در مسیر پوشه پروژه باز کنید
3. کتابخانه‌ها را نصب کنید:
   pip install -r requirements.txt
4. برنامه را اجرا کنید:
   python house_price_prediction_extra.py

📊 مراحل اجرای پروژه:
1. بارگذاری و بررسی داده‌ها
2. تحلیل اکتشافی (EDA)
3. پیش‌پردازش داده‌ها
4. آموزش مدل رگرسیون خطی
5. ارزیابی مدل
6. مصورسازی نتایج

📌 نتایج مدل:
- RMSE: 68328.42
- مهم‌ترین ویژگی‌های تاثیرگذار:
  1. median_income (درآمد خانوار)
  2. housing_median_age (سن خانه)
  3. total_rooms (تعداد اتاق‌ها)
